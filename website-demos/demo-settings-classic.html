FAQ
完整版和精简版有什么区别? 完整版内置 SenseVoice + Qwen3 本地模型，无需网络即可识别。精简版需要配置云端供应商 API。
Intel Mac 能用吗? 可以，但本地模型中 Qwen3-ASR 需要 Apple Silicon。Intel 用户建议使用云端识别或仅 SenseVoice。
如何添加自定义文本处理? 在设置的「处理模式」中新建模板，填入 prompt 即可。支持任何兼容 OpenAI API 的 LLM 供应商。
