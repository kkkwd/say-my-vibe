# Type4Me v1.9.0 Release Test Cases

## 1. 基础录音与注入

- [ ] 按住快捷键录音，松开后文字注入到光标位置
- [ ] 录音过程中浮窗显示实时识别文本
- [ ] 短录音 (<0.5s) 不产生幻觉文字
- [ ] ESC 打断录音，文字不注入但保留在剪贴板

## 2. 剪贴板行为 (#108, #111, #107)

- [ ] 录音前剪贴板有内容 A，录音说 B，注入 B 后剪贴板恢复为 A
- [ ] 注入失败时（非文本输入框），识别文字保留在剪贴板供手动粘贴
- [ ] ESC 打断后，剪贴板内容是识别文字（不是旧内容）
- [ ] 关闭"复制到剪贴板"选项后，剪贴板不被覆盖

## 3. 文本注入兼容性

- [ ] 注入到 VS Code (Electron)
- [ ] 注入到微信输入框
- [ ] 注入到 Safari 地址栏
- [ ] 注入到 Spotlight 搜索栏
- [ ] 注入到 Terminal

## 4. 蓝牙音频 (新功能)

- [ ] 蓝牙耳机/音箱连接时，开始提示音完整播放（不截断前几百 ms）
- [ ] 音箱保活开关：打开后 BT 音箱不休眠断开
- [ ] 麦克风保活开关：打开后 BT 麦克风不断开
- [ ] 切换提示音输出设备后，提示音从新设备播放

## 5. 麦克风选择 (#113)

- [ ] 设置中可选择指定麦克风设备
- [ ] 选择后录音使用指定设备（不是系统默认）
- [ ] 设备拔出后自动回退到系统默认

## 6. 鼠标按键快捷键 (#113)

- [ ] 可在设置中录制鼠标中键作为录音快捷键
- [ ] 鼠标中键按住录音，松开停止
- [ ] 鼠标侧键同理
- [ ] 快捷键显示正确（"鼠标中键" / "鼠标按键 4" 等）

## 7. 提示音

- [ ] 所有提示音选项可正常播放预览（电子提示音 / 拨弦 / 沉浸 / 乒 / 水滴 / 键盘）
- [ ] "清脆双音"选项已移除，不在列表中
- [ ] 已选"清脆双音"的旧用户，升级后自动回退到"电子提示音"

## 8. 订阅剥离验证

- [ ] 设置侧边栏无"账户"选项
